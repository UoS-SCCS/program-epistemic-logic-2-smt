Description

The module define the logical languages and everything that is needed inside them * agents and variables * expressions (boolean and integer expressions), * mathcal{L_{DK}}, mathcal{L_{square K}}, mathcal{L}_{FO} formulas * programs inside the box * weakest preconditions, strongest postconditions * substitutions in expressions and formulas

See also the companion paper here https://doi.org/10.48550/arXiv.2206.13841 Maintainer: S.F. Rajaona sfrajaona@gmail.com

eBVar constructs an existential boolean variable \(xb_i\) for an integer i

uBVar :: Int -> Var #

uBVar constructs an universal boolean variable \(ub_i\) for an integer i

eIVar :: Int -> IntDomain -> Var #

eBVar constructs an existential integer variable \(xi_i\) for an integer i

uIVar :: Int -> IntDomain -> Var #

uBVar constructs an universal integer variable \(ui_i\) for an integer i

forAllB :: (Var -> Formula Modal) -> Formula Modal #

this function chooses an appropriate index \(i\) to build universal quantification \(\exists y_i \ldots\) from of a lambda abstracted ModalFormula.

existsB :: (Var -> Formula Modal) -> Formula Modal #

similar to forAllB but with existential
